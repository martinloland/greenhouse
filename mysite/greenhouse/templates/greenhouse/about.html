About

"An experiment in django web development and automatic control of plants using python and raspberry pi"


{% include 'greenhouse/includes/carousel.html'%}

I have have always been fascinated by home automation and this project is an investigation of what could be possible using a raspberry pi, some basic electronics and the magic of python.

Technology and hardware

  • Django web framework
  • Built in SQLite database
  • Raspberry pi zero with built in WiFi
  • Raspberry Pi camera v3
  • nginx HTTP server
  • uWSGI for web server gateway interface

When the raspberry pi boots it will automatically start two processes. One starts the server and the other is a python scripts that read sensor values, captures images and control the lights and water pump. Both processes communicate with a local database file to write sensor values, read settings etc.

The current version is in debug mode, this means that the video feed and sensor readings are not taken from an actual camera or sensor.
